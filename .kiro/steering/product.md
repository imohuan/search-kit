# 产品概览

## 项目名称
文档智搜 Pro (Document Smart Search Pro)

## 项目目的
一个基于 Vue 3 + TypeScript 的智能文档搜索、提取和管理 Web 应用。用户可以上传 DOCX 和 PDF 文档，进行高级全文搜索、提取特定文本模式，并维护已处理文档的库。

## 核心功能
- **文档库管理**：上传和管理 DOCX、PDF 文件
- **智能搜索**：全文搜索，支持精确匹配、最大间隔容差等高级选项
- **文本提取**：从文档中提取特定字符/模式，提供可视化网格界面
- **文档解析**：将文档转换为可搜索的纯文本和 HTML 格式
- **本地存储**：所有文档存储在 IndexedDB 中，支持离线访问

## 目标用户
需要快速搜索和提取多个文档中信息的专业人士。

## 核心模块
- **搜索模块**：全文搜索，支持片段预览
- **库管理模块**：文档上传和管理
- **提取模块**：字符/模式提取，提供可视化界面
